--- title: How to forward Ports layout: docs category: usage applicable_to: 2.x,3.x ---

The primary functionality of Beacon is to expose ports for dedicated server applications (such as game servers). All related options are directly available from the primary application window:

The window consists of the following three primary components:

  1. Service List - This list displays all applications for which port definitions are provided
  2. Service Actions - The open and close port actions will appear here
  3. Port List - This list displays all ports for a selected application
  4. External Address - This is the IP address at which exposed applications will become available

Step by Step

  1. Start the server (a dedicated Minecraft Server, for instance)
  2. Select an application ("Minecraft: Java Edition", for instance) from the service list on the left
  3. Select "Open Ports"
  4. Copy the external address ("10.42.42.1" in the example above) and forward it to your friends/family/etc (whoever you wish to let into your server)
  5. (Optional) Minimize the application (By default, it will appear in your system tray (see below) and may be restored via double click)
  6. (Optional) Once you are done playing, press the "Close Ports" button
Important
Beacon needs to remain running while ports are exposed. Closing the application will also close all previously opened ports.

System Tray

Unless explicitly disabled, Beacon will place itself within your System Tray when minimized. It will remain running in the background and keep your ports open within this mode. In order to restore it, double click the Beacon icon.

This functionality can be disabled within the application settings.

For Windows Users
By default, Beacon will be hidden within the extended Tray Menu (accessible through the up arrow located directly next to the tray area). If desired, this may be changed via the Taskbar Settings menu.
