--- title: Custom Repositories layout: docs category: usage applicable_to: 3.x ---

As of version 3.0 of Beacon, users may configure their own service repositories in addition to the stock repositories in order to share their definitions with other people. The user repository list is available as part of the General page within the application settings:

  1. Repository List - Displays the URIs for each configured user repository
  2. Add Repository - Opens the repository editor for a new repository
  3. Remove Repository - Removes the selected repository

By default, no user repositories are configured when launching the application.

Step by Step

  1. Retrieve the repository URI from its maintainer
  2. Access the settings via the Settings button in the main window
  3. Select "Add Repository"
  4. Enter the repository URI
  5. Select "Save"
  6. Restart the application
